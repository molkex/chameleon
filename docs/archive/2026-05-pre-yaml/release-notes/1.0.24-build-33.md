# What's New — 1.0.24 (33)

## RU (для русской локализации)
Подключение работает быстрее и стабильнее:
• Когда вы выбираете страну, приложение сразу проверяет какой протокол реально работает в вашей сети, и выбирает его. Раньше могла «прилипнуть» комбинация, не работающая на текущем интернете.
• Если все серверы выбранной страны недоступны — автоматически переключаем на следующую страну с лучшим пингом.
• Если все обычные страны недоступны — последний рубеж: маршрут через российский relay.
• Когда сервер не отвечает, мы тихо сообщаем об этом нашим серверам, чтобы быстрее починить.

## EN (default English)
Faster and steadier connection:
• When you pick a country, the app now immediately tests which protocol actually works on your current network and uses it. Before, it could stick with a stale pick that didn't work on this connection.
• If every server in your chosen country is down, we automatically switch to the next country with the best ping.
• If all regular countries are down, we use a Russian bypass relay as a last resort.
• When a server stops responding, we quietly tell our backend so we can fix it faster.
