# What's New — 1.0.24 (32)

## RU (для русской локализации)
Умное восстановление и упрощённый выбор стран:
• Если выбранная страна перестаёт отвечать, приложение само переключит вас на рабочий сервер — трафик не прервётся.
• Тап по стране в списке сразу её выбирает, без вложенного меню. Лучший протокол выбирается автоматически.
• На главном экране теперь видна только страна — чисто и понятно, как у крупных VPN.
• Новый переключатель «Авто-восстановление» в Настройках → Маршрутизация (включён по умолчанию).

## EN (default English)
Smart recovery and a simpler country picker:
• If your chosen country stops responding, the app automatically switches you to a working server — your traffic keeps flowing.
• Tap a country in the list to pick it instantly, no submenu. The best protocol is chosen automatically.
• The home screen now shows just the country — clean and clear, like top VPN apps.
• New "Auto-recover" toggle in Settings → Routing (on by default).
