# What's New — 1.0.24 (34)

## RU (для русской локализации)
Исправлена ошибка из билда 33:
• Когда вы пинуете страну, приложение теперь действительно сразу проверяет какой протокол работает в текущей сети — и переключается на рабочий за ~3 секунды. В прошлом билде проверка пропускалась, и могла залипнуть нерабочая связка (например, прямой DE на мобильном интернете).

## EN (default English)
Fixes a bug from build 33:
• When you pin a country, the app now actually re-probes which protocol works on the current network and switches to it within ~3 seconds. Previously the probe was skipped, so a dead leg (e.g. direct DE on mobile data) could remain selected.
