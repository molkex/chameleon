Implementation Handoff

Спецификация для сборки в SwiftUI. Это направление рассчитано на минимум экранов и максимально прямой сценарий пользователя.

Layout Rules

Safe areasTop content starts after status bar + 12pt.
Horizontal padding16pt for root screens, 12pt in cards.
Spacing scale8 / 12 / 16 / 24 / 32 only.
Corner radiiCard 13pt, sheet 16pt, primary orb 87pt radius.

Behavior Contracts

Connect CTAAlways visible on Home; tap toggles connection state.
Route changeImplemented as bottom sheet, no separate locations tab.
Subscription stateHome and Account show same expiry source of truth.
Error surfacesOne primary action + one secondary action maximum.

Animation Rules

State transitionsUse 220ms easeOut for card and badge updates.
Orb pulse2.8s cycle, disabled when Reduce Motion enabled.
SheetSpring 0.32 / damping 0.86 to match iOS modal feel.

QA Checklist

AccessibilityAll touch targets >= 44pt and text contrast >= 4.5:1.
State coverageConnected / disconnected / connecting / no internet / expired.
NavigationOnly two tabs for this direction: Home and Account.
LocalizationAll labels fit RU/EN strings without clipping.
