Chameleon iOS Design Systems V3

Переделал с нуля по мобильному фокусу: выраженная визуальная система, нормальная иерархия, читаемые touch-паттерны и продуманный user-flow. Смотри каждое направление отдельно.

Direction 01

Premium Minimal

Спокойный premium-стиль. Минимальная IA: 2 экрана (`Home + Locations`) и smart sheet для остального.

 Direction 02

Tech / Pro

Больше оперативной информации: health, route stability, задержка и быстрые технические действия.

 Direction 03

Bold / Brand

Сильный визуальный характер для consumer-позиционирования: ярко, но с дисциплиной в UX.
Для минимума экранов: бери `Direction 01`. Там уже заложен быстрый сценарий: открыть → connect → при необходимости сменить локацию.
