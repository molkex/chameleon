# Chameleon VPN ‚Äî engine architecture

The live VPN data plane: how a packet leaves the iPhone, what config the backend
generates for client and server, and the rules that keep the tunnel evading RU DPI.
**Why** VLESS Reality + sing-box 1.13 (vs WireGuard/OpenVPN/Xray-client) is in
[`../decisions/0002-singbox-1.13-vless-reality.md`](../decisions/0002-singbox-1.13-vless-reality.md) ‚Äî
not repeated here. For topology/overview see [overview.md](overview.md).

Code is source of truth. Anchors:
[`clientconfig.go`](../../backend/internal/vpn/clientconfig.go) (iOS config),
[`singbox.go`](../../backend/internal/vpn/singbox.go) (server config + engine),
[`engine.go`](../../backend/internal/vpn/engine.go) (Engine interface + types),
[`userapi.go`](../../backend/internal/vpn/userapi.go) (User API client).

## Packet path (iPhone ‚Üí exit ‚Üí internet)

```
App traffic
  ‚îî‚îÄ TUN inbound  (utun, 172.19.0.1/30, MTU 1400, stack:"system", auto_route)
       ‚îî‚îÄ sing-box (libbox 1.13 inside PacketTunnel NE)
            ‚îú‚îÄ route.rules (ORDER MATTERS ‚Äî see below)
            ‚îú‚îÄ DNS: dns-remote (1.1.1.1, detour:"Proxy") + dns-direct (77.88.8.8 Yandex, .ru/banks) + fakeip
            ‚îî‚îÄ outbound selectors ‚Üí ONE pre-selected leaf
                 ‚îî‚îÄ VLESS Reality TCP + xtls-rprx-vision + uTLS (per-user fingerprint)
                      ‚îú‚îÄ direct exit   ‚Üí NL 147.45.252.234:443  /  GRA 54.38.243.162:443
                      ‚îú‚îÄ relay chain   ‚Üí MSK 217.198.5.52 (vless ‚Üí userspace WG ‚Üí NL/GRA)
                      ‚îî‚îÄ whitelist     ‚Üí SPB 185.218.0.43 (RU whitelist-bypass legs)
                 ‚Üí Xray-core 25.12.8 inbound on the exit ‚Üí direct outbound ‚Üí internet
                    (NL egress source-bound to 72.56.79.25 for clean US geo)
```

Leaf selection happens in the iOS main app's `PathPicker` (NWConnection probes in
the host process), then the extension receives ONE leaf pre-selected ‚Äî this avoids
urltest Go-heap spikes that risked the ~50 MB NetworkExtension jetsam cap.

## Client config shape (what `clientconfig.go` generates)

Per-user JSON, fetched via `GET /api/v1/mobile/config`. Leaf tag format
`{cc}-{kind}-{key}` (kind = `direct|h2|tuic|via`); iOS derives the country from the
prefix. Outbound topology:

- **`Proxy`** (selector, default `Auto`) ‚Äî top-level user choice. Members:
  `Auto`, per-country urltest groups, individual leaves, the whitelist group.
- **`Auto`** (urltest) ‚Äî all standard leaves, probe `gstatic.com/generate_204`,
  `interval:"10s"`, `tolerance:0`, `interrupt_exist_connections:true` (build-39/40).
- **Per-country urltests** (`üá≥üá± –ù–∏–¥–µ—Ä–ª–∞–Ω–¥—ã`, `üá´üá∑ –§—Ä–∞–Ω—Ü–∏—è`, ‚Ä¶) ‚Äî strict
  single-country, no cross-country failover (build-42: picking a country means
  "exit there or fail").
- **`üá∑üá∫ –Ý–æ—Å—Å–∏—è (–æ–±—Ö–æ–¥ –±–µ–ª—ã—Ö —Å–ø–∏—Å–∫–æ–≤)`** ‚Äî `selector` (NOT urltest) over the SPB
  whitelist-bypass legs; manual-only, excluded from `Auto`.
- **Mode selectors** `RU Traffic` / `Blocked Traffic` / `Default Route` ‚Äî flipped
  together via Clash API to switch smart / ru-direct / full-vpn without reconnect.
- **Leaf outbounds**: VLESS Reality (always), plus Hysteria2/TUIC v5 *only when the
  exit ships a pinnable UDP cert* (`tls.certificate` PEM ‚Äî SEC-03, never `insecure:true`).
- **System**: `direct`, `block`.

Route rule-sets pulled remotely: `geoip-ru` (SagerNet) + `refilter` (RKN block list).

## Server config shape (what `singbox.go` builds)

Generated by `buildServerConfig()` on each exit/relay, validated with
`sing-box check` before the atomic rename:

- **Inbound** `vless-reality-tcp`: `listen:"::"`, `listen_port` (prod 443),
  users `{name, uuid, flow:"xtls-rprx-vision"}`, TLS Reality
  (`server_name` = SNI, `handshake ‚Üí {SNI}:443`, `private_key`, `short_id:[‚Ä¶]`).
- Optional Hysteria2 (`salamander` obfs) + TUIC v5 inbounds ‚Äî only when
  `UDPCertPath` set (prod UDP ports currently unset).
- **Outbound** `direct` (`inet4_bind_address` = `EgressBindIP`, NL ‚Üí 72.56.79.25).
- **Services**: `user-api` on `127.0.0.1:15380`. **Experimental**: `clash_api`
  (127.0.0.1:9090, stats) + `v2ray_api` (gRPC, per-user traffic).
- Reality private key lives in the DB (`vpn_servers`), read at engine start ‚Äî one
  source of truth, no `.env` key files.

## User API (zero-downtime user management)

REST on `127.0.0.1:15380` (sing-box fork), `Authorization: Bearer ${USER_API_SECRET}`.
Engine calls it first; on failure falls back to config-rewrite + SIGHUP. After a
successful API call the engine also persists the config to disk (rollback on write
failure) so runtime never diverges from disk (audit MED-002).

```
POST   /api/v1/inbounds/{tag}/users        add    {name, uuid, flow}
DELETE /api/v1/inbounds/{tag}/users/{name} remove
PUT    /api/v1/inbounds/{tag}/users        bulk replace {users:[‚Ä¶]}
GET    /api/v1/inbounds/{tag}/users        list
```

The same client (`NewUserAPIClientFromURL`) drives **`RelayUserSyncer`**
([`../../backend/internal/cluster/relay.go`](../../backend/internal/cluster/relay.go)):
NL pushes the active user set to remote exits/relays (GRA `user_api_url`
`http://54.38.243.162:15380`) ‚Äî event-driven on each mutation + a periodic safety
net. The User API is ufw-restricted to NL + bearer-gated.

## Route rule ORDER (client) ‚Äî must not reorder

```jsonc
[
  {"action":"sniff"},                                          // 1. protocol detect ‚Äî MUST be first
  {"protocol":"dns","action":"hijack-dns"},                    // 2. capture DNS (hyphen spelling)
  {"clash_mode":"Direct","outbound":"direct"},                 // 3. global bypass switch
  {"network":"udp","port":443,"action":"reject","no_drop":true}, // 4. kill QUIC (no_drop or iOS hangs)
  {"ip_is_private":true,"outbound":"direct"},                  // 5. LAN never tunnelled
  {"domain_suffix": ruAlwaysDirect, "outbound":"direct"},      // 6. RU banks/gov direct
  {"rule_set":"refilter","outbound":"Blocked Traffic"},        // 7. RKN-blocked ‚Üí VPN
  {"domain_suffix":[".ru"],"outbound":"RU Traffic"},           // 8. .ru ‚Üí mode selector
  {"rule_set":"geoip-ru","outbound":"RU Traffic"}              // 9. RU geoip ‚Üí mode selector
]
```
`final: "Default Route"`. `default_domain_resolver` = `dns-remote` (detour:Proxy) so
proxied DNS geolocates to the exit country, not RU.

## Critical rules

| Rule | Detail |
|---|---|
| **Reality SNI** | ONLY vetted, non-blocked domains. **Never** google.com / cloudflare.com. Current `ads.adfox.ru`. New SNI ‚Üí check RKN block list first. |
| **short_id** | Empty `""` is always valid. A random short_id not in the server set ‚Üí `reality verification failed`. |
| **route order** | `sniff` first, then `hijack-dns`. Wrong order ‚Üí DNS loop. |
| **DNS in 1.13** | DNS servers route direct by default ‚Äî no `detour:"direct"` needed (use detour only to force a path, e.g. `Proxy`). |
| **QUIC reject** | `no_drop:true` on the UDP/443 reject, or iOS hangs on some sites. |
| **TUN stack** | `"system"` only ‚Äî `mixed`/`gvisor` break the NetworkExtension. |
| **mux** | sing-box `mux` (h2mux) is **incompatible** with the Xray server ‚Äî never enable on the client. |
| **Xray version** | Server-side Xray-core pinned **25.12.8**. **NOT v26** ‚Äî incompatible with sing-box 1.13. |
| **UDP TLS** | Hysteria2/TUIC legs ship only with a **pinned** cert (`tls.certificate`), never `insecure:true` (SEC-03). |
| **validate** | Always `sing-box check -c config.json` before any rollout (also enforced in `writeConfigLocked`). |
| **migration** | On any sing-box version bump, read https://sing-box.sagernet.org/migration/ before regenerating configs. |
