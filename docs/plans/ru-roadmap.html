02

Стратегические опоры

Три темы, в которые группируются все инициативы. Каждая инициатива ниже привязана к опоре (цветной маркер слева).

Опора A

Диверсификация каналов

Не зависеть от одного магазина: iOS-витрина, Android/RuStore, прямая продажа.

Опора B

Рост конверсии

Выжать короткое окно жизни: онбординг, оплата без трения, офферы.

Опора C

Защита основного бизнеса

Изоляция основного аккаунта Apple от рискованных экспериментов.

09

Допущения и зависимости

Допущения
  • Окно жизни iOS-публикации ~2 суток сохранится
  • RuStore продолжит терпеть VPN-приложения
  • Бэкенд и протокол переиспользуются на Android
  • Есть ИП для регистрации RuStore-аккаунта
Зависимости
  • I-05 (RuStore) ← I-04 (Android-клиент)
  • I-07 (конвейер) ← I-05 (первая публикация)
  • I-08 (Google Play) ← I-04 (Android-клиент)
  • Воронка v1 (I-03) питает данные для I-06
