# What's New — 1.0.24 (35)

## RU (для русской локализации)
Подключение работает почти мгновенно даже на нестабильных сетях:
• Перед подключением приложение само проверяет какой сервер реально работает в вашей сети, и сразу заходит через него — никаких больше «3 минуты ждать пока починится».
• Если сервер вдруг перестал работать в момент использования — переключение на следующий рабочий за ~5 секунд (раньше до 3 минут).
• Приложение запоминает какой сервер хорошо работал на этой WiFi/мобильной сети, и в следующий раз заходит через него мгновенно.

## EN (default English)
Connection is now near-instant even on unreliable networks:
• Before connecting, the app probes which server actually works on your current network and goes through it immediately — no more "wait 3 minutes for it to settle".
• If the server stops working mid-session, the app switches to the next working one within ~5 seconds (was up to 3 minutes).
• The app now remembers which server worked best on this WiFi / cellular network and reconnects through it instantly next time.
