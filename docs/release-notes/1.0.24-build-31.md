# What's New — 1.0.24 (31)

## RU (для русской локализации)
Стабильность и быстрый сервер-пикер:
• Расширение туннеля больше не выключается через 3–5 минут на мобильном интернете (исправлен лимит памяти 50 МБ).
• Выбор сервера сохраняется при переподключении — раньше после смены сети могло возвращаться к Auto.
• Точные пинги на UDP-протоколах (Hysteria2, TUIC) — больше не "3 мс" по ошибке.
• Прямой выбор протокола в Германии и Нидерландах: Reality, Hysteria2, TUIC, через MSK.

## EN (default English)
Stability and a faster server picker:
• The VPN tunnel no longer dies every 3–5 minutes on cellular (50 MB memory cap fix).
• Your server choice now survives reconnects — previously it could revert to Auto after a network change.
• Real ping values for UDP-based protocols (Hysteria2, TUIC) — no more fake "3 ms".
• Pick a specific protocol per country in Germany and Netherlands: Reality, Hysteria2, TUIC, via MSK.
