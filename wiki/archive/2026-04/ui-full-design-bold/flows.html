UX Flows

Флоу в этом направлении построены вокруг одного эмоционального центра: мгновенное ощущение контроля и защищенности через главный экран.

Flow 01: First Launch

goal: strong first impression + fast permission completion
1. Onboarding with brand mood and clear trial offer.
2. VPN permission request immediately after primary CTA.
3. User lands on Home disconnected state.
KPI: first-session activation > 80%.

Flow 02: Connect Loop

goal: 1-tap connection and clear status feedback
1. Tap orb to connect.
2. Connecting screen with progress context.
3. Connected state shows route/session/subscription.
4. Tap orb to disconnect when needed.
KPI: recurrent connect usage daily active users > 60%.

Flow 03: Route Selection

goal: high clarity in a brand-heavy UI
1. Open Locations tab from bottom nav.
2. Select route by region and latency.
3. Apply route and return to Home.
KPI: successful route switch > 95%.

Flow 04: Subscription Recovery

goal: recover expired users without friction
1. Expired state appears with clear consequence message.
2. Paywall offers renew and restore.
3. On success, user returns to prior route context.
KPI: renewal conversion from expired state > 30%.

Flow 05: Error Recovery

goal: preserve trust during failures
1. No-internet screen explains the issue in plain language.
2. Retry and diagnostics actions are visible above fold.
3. Escalate to support with diagnostic copy if needed.
KPI: self-recovery within app > 70%.
