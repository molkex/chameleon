Implementation Handoff

Контракт для iOS реализации технического направления. Важна предсказуемость данных и консистентность state-machine во всех экранах.

Data + State Contracts

Single source of truth for tunnel state: disconnected / connecting / connected / incident / expired.
Metrics block updates by timer (1s) only while connected; paused in background.
Locations list sorted by RTT, with pinned node surfaced on top.
Diagnostics report generated as JSON and share-sheet ready.

Layout + Components

Base padding: 16pt. Card padding: 11pt. Row min height: 44pt.
Terminal cards use JetBrains Mono 12/1.52 for dense metrics.
Bottom navigation has exactly 4 tabs in this direction.
Incident banner stays sticky until retry outcome is known.

Motion + Interaction

Connect orb animates with subtle glow in connected state; no infinite heavy loops.
Screen transitions 160-220ms ease-out; diagnostics refresh animates with crossfade.
Reduce Motion: replace animated orb with static icon + textual state badge.

QA Checklist

All interactive controls >= 44x44 pt.
Color contrast passes 4.5:1 for text and 3:1 for UI elements.
Incident flow tested offline and with simulated API timeout.
Exported diagnostics report includes timestamp, route, metrics, app version.
