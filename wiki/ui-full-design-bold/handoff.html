Implementation Handoff

Спецификация для iOS разработки брендового направления. Важно сохранить эмоцию интерфейса и при этом не потерять функциональную строгость.

Layout Rules

Base horizontal padding 16pt, card internal padding 11pt.
Primary orb fixed size 182x182 on compact width.
Bottom nav always visible on Home/Locations/Account.
Large title block max 2 lines; hard truncation after line 2.

Interaction Contracts

Orb tap toggles connect/disconnect; haptic feedback on state change.
Locations list supports quick apply without confirmation modal.
Subscription warning pill appears when <= 3 days left.
Expired and no-internet states must expose retry path immediately.

Motion Rules

Orb float animation: 3.1s ease-in-out, amplitude 5px.
Card transitions: 220ms ease-out for status updates.
Reduce Motion: disable orb float and glow pulsation.

QA Checklist

Touch targets >= 44pt on all actionable rows and buttons.
Contrast validated for gradient over dark surfaces.
State matrix covered: connected, connecting, disconnected, expired, incident.
Brand gradients render consistently on iPhone SE to Pro Max widths.
